== Description ==

Google Locker is a plugin help you to get more Share/Subscribe, traffic and customers.

### Key Features

* Google+ Button, Google Share, Youtube Subscribe.
* Locker Types: Social, Content with Shortcode.
* Locker Styles: Start, Corner, Round.
* Effects for social button with behavior on click or hover.
* Supported 2 Social Layouts.
* Language setting for Buttons.
* Unlimited Color, Text, ... and number of content lockers.
* Preview on Plugin Options page.
* Powerful tinyMCE toolbar plugin.
* Cookies remember users who have already unlocked.
* Multiple Lockers on the same Page and Post.
* Powered by jQuery (noConflict mode).

### Premium Features (Secure Social Locker)

* Facebook Like & Share support.
* Twitter Tweet & Follow support.
* LinkedIn support.
* SEO friendly.
* Order Social Buttons.
* Tracking & Reporting statics on Plugin Options page.

== Installation ==

1. Login to WordPress Admin panel, go to 'Plugins', next click 'Add New', choose 'Upload', click 'Browse' and find a zip plugin file and next click 'Upload Now'

2. If plugin is installed, go to 'Settings' choose 'Google Locker' and now you can configure the plugin.

== How to use ==

1. Select the content in the post editor that you want to lock inside.

2. Click the lock icon. You can choose Social, Content button to insert ShortCode to content locker.
